# balenaBlocks built for balenaCloud

balenaBlocks are pre-built container images which developers can add to their multicontainer apps to provide useful functionality. Their aim is to reduce the friction of creating IoT applications which enables rapid prototyping and development. These intelligent, drop-in chunks of functionality are built to handle the basics, allows you as developers to focus on solving the hard problems. You can read more about balenaBlocks on our [blog](https://www.balena.io/blog/introducing-balenablocks-jumpstart-your-iot-app-development/).

Checkout [balenaBlocks](/learn/develop/blocks/) docs to build & publish your first block on balenaHub. To find the complete list of available blocks to try and deploy, head to [balenaHub]({{ $links.balenaHubUrl }}/blocks)!
